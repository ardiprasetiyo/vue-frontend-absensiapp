# vue-frontend-absensi

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Build to Android Apps using Capacitor
```
npx cap copy
npx cap update
npx cap open android
```
